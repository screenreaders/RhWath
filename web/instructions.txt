TEST RHVoice NA ZEGARKU (Wear OS) - KROK PO KROKU

1) Wlacz opcje programistyczne na zegarku
- Ustawienia -> Informacje o zegarku -> Informacje o oprogramowaniu
- Stuknij "Numer kompilacji" 7 razy
- Wroc do Ustawienia -> Opcje programistyczne

2) Wlacz debugowanie ADB
- Opcje programistyczne -> ADB debugging (wlacz)
- Wlacz Wireless debugging / Debugowanie przez Wi-Fi

3) Polacz ADB z zegarkiem
Wersja A (parowanie):
- Wireless debugging -> Pair new device
- Na komputerze:
  adb pair IP:PORT
  # wpisz kod z zegarka
  adb connect IP:PORT_Z_ADB
  adb devices

Wersja B (starsze debugowanie Wi-Fi):
- Na zegarku zobaczysz IP i port (zwykle 5555)
- Na komputerze:
  adb connect IP:5555
  adb devices

4) Zainstaluj APK (uruchom w folderze z pobranymi plikami lub podaj pelna sciezke)
adb install -r rhvoice-core.apk
adb install -r rhvoice-Polish.apk
adb install -r rhvoice-Cezary.apk

5) Ustaw RHVoice jako domyslny TTS
- Ustawienia -> Ulatwienia dostepu -> Syntezator mowy
  albo: Ustawienia -> System -> Jezyki i wprowadzanie -> Syntezator mowy
- Wybierz RHVoice
- Ustaw jezyk Polski i glos Cezary

6) Test dzwieku
- W ustawieniach TTS kliknij "Odtworz przyklad"
- Jesli slychac glos Cezarego - dziala

7) Logi (gdyby byly problemy)
adb logcat | grep -i rhvoice
