RHVoice Watch

Lekki silnik TTS dla zegarków. Pobierz APK, przeczytaj instrukcję i uruchom głos na Samsung Galaxy Watch 8 Classic.

Pobierz APK Instrukcja
Docelowe urządzenie Galaxy Watch 8 Classic
Architektura arm64-v8a
APK Podpisane cyfrowo

Szybki start

  1. Pobierz i zainstaluj rhvoice-core.apk
  2. Pobierz i zainstaluj rhvoice-Polish.apk
  3. Wybierz głos: rhvoice-Cezary.apk
  4. Ustaw RHVoice jako domyślny TTS
ADB jest wymagane do instalacji na zegarku. Szczegóły poniżej.

Kolejność instalacji

              
Core -> Polish -> wybrany głos. Bez core i danych języka głos nie ruszy.

            

            

              
Bezpieczne ADB

              
Po testach wyłącz ADB debugging w opcjach programistycznych.

            

            

              
Wydajność

              
Wyłącz nieużywane głosy, aby oszczędzić pamięć i miejsce.
